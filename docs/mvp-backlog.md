# Backlog MVP — Joaquín Studio Salon

## Épica 1: Autenticación y acceso
- [ ] Login con Supabase Auth (email + password)
- [ ] Persistencia de sesión
- [ ] Middleware de protección de rutas
- [ ] Roles básicos (admin, estilista)

## Épica 2: Dashboard
- [ ] Tarjetas de KPIs (hoy, próximas, clientes, ingresos)
- [ ] Lista de citas del día
- [ ] Accesos rápidos a alta de cliente/cita

## Épica 3: Clientes y expediente
- [ ] Alta/edición de cliente
- [ ] Búsqueda por nombre/teléfono
- [ ] Vista de expediente técnico
- [ ] Carga de fotos antes/después

## Épica 4: Agenda y citas
- [ ] Alta/edición/cancelación/confirmación de cita
- [ ] Vista diaria y semanal
- [ ] Estados: pendiente, confirmada, en_proceso, finalizada, cancelada
- [ ] Link de WhatsApp prellenado

## Épica 5: Servicios, visitas y fórmulas
- [ ] Catálogo de servicios con precio base y duración
- [ ] Registro de visita con costo final
- [ ] Registro de fórmulas (producto, proporción, volumen, tiempo)
- [ ] Sugerencia de próxima cita

## Criterios de salida MVP
- [ ] Flujo completo cliente nuevo funcionando de punta a punta
- [ ] Flujo cliente existente funcionando
- [ ] Evidencia de uso en móvil y desktop
